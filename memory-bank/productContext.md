# Product Context - 동유기 (JTE)

## 제품 비전

동유기는 환상서유기의 정신을 계승한 모바일 2D RPG입니다. 삼장법사 일행의 서역 여정을 배경으로, 탐험과 전투, 유머러스한 스토리가 어우러진 게임입니다.

## 해결하는 문제

1. **모바일 RPG의 피로감**: 복잡한 조작과 긴 플레이 세션
   - 동유기: 터치 전용 단순 조작, 짧은 세션 설계

2. **스토리와 게임플레이의 분리**: 스토리는 읽기만 하고 플레이는 별개
   - 동유기: 탐험 중 대화, 꽁트 연출, 퍼즐이 자연스럽게 연결

3. **모바일 전투의 단순함**: 오토 배틀 위주
   - 동유기: 칸 기반 전략 전투로 깊이 있는 전술 제공

## 핵심 사용자 경험

### 게임 흐름
```
부팅 → 타이틀 → 애니메이션(컷신) → 탐험 → 전투 → 탐험
```

### 탐험 (Explore)
- **이동**: 탭하여 목적지로 이동
- **대화**: NPC 탭하여 대화 시작
- **조사**: 오브젝트 탭하여 상호작용
- **발견**: 숨은 상자, 퍼즐, 이벤트
- **전투 진입**: 적 심볼 접촉 또는 이벤트

### 전투 (Battle)
- **칸 기반**: 유닛이 칸에 배치
- **턴제**: 속도 기반 행동 순서
- **전략**: 스킬 범위, 위치 이동, 상태이상
- **UX**: 캐릭터 선택 → 행동 선택 → 칸 선택

### 애니메이션 (Animation)
- **자동 진행**: 컷신 재생
- **빠른 진행**: 화면 누르고 있으면 가속
- **탐험 전환**: 종료 후 즉시 조작권 반환

## 사용자 페르소나

### 주 타겟: 20-40대 한국/아시아 게이머
- 환상서유기, 한국형 RPG에 향수
- 모바일 게임 선호
- 스토리와 전략 모두 중시

### 서브 타겟: 글로벌 RPG 팬
- 동양 판타지에 관심
- 턴제 전략 게임 경험

## 모바일 UX 원칙

1. **터치 직접 조작**: 가상 조이스틱 없음
2. **자동 접근 상호작용**: 오브젝트 탭 시 자동 이동 후 실행
3. **자주 이어하기**: 자동 저장, 중단 저장
4. **명확한 예고**: 행동 선택 시 영향 범위 미리 표시
5. **실수 복구**: 쉬운 취소, 되돌리기

## 국제화 전략

### 지원 언어
- 한국어 (1차)
- 중국어, 일본어 (2차)
- 영어 (3차)

### 현지화 원칙
- 번역 키 기반 텍스트 관리
- 언어별 텍스트 길이 대응
- 문화적 맥락 고려 (꽁트 대사)

## 차별화 요소

1. **칸 기반 전투**: 스킬이 유닛이 아닌 칸 패턴 기준
2. **팀 분기 시스템**: 한국 팀 / 일본 팀 분리 플레이
3. **꽁트 스타일 스토리**: 유머러스한 대화와 상황
4. **탐험 중심**: 전투가 탐험의 일부로 자연스럽게 연결
5. **모바일 최적화**: 터치 UX, 세션 설계

## 성공 지표

- 파트 1 완료율
- 일일 플레이 세션 수
- 전투 승률 및 재시도
- 언어별 리텐션
