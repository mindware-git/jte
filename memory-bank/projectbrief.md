# Project Brief - 동유기 (JTE)

## 프로젝트 개요

**동유기(JTE - Journey to the East)**는 환상서유기 스타일의 모바일 2D RPG입니다. 삼장법사와 그의 일행이 서역으로 떠나는 여정을 다룬 동양 판타지 게임입니다.

## 핵심 목표

1. **환상서유기식 게임플레이**: 탐험, 전투, 퍼즐, 스토리 연출의 조화
2. **모바일 최적화**: 터치 전용 UX, 짧은 세션 플레이 지원
3. **국제화**: 한국어, 중국어, 일본어, 영어 지원
4. **AI 우선 개발**: Spec → Test → Implementation 흐름

## 플랫폼

- **엔진**: Godot 4
- **배포 대상**: Android, iOS
- **해상도**: 모바일 세로 화면 기준

## 핵심 재미 요소

- **탐색**: 마을, 던전, 필드 탐험
- **반복 대화**: NPC와의 유머러스한 대화
- **숨은 상자**: 탐색 보상
- **퍼즐**: 이동과 조사 기반 기믹
- **꽁트 컷신**: 스토리 연출

## 주요 캐릭터

- **손오공**: 전열 돌파형, 직선/돌진형 칸 공격
- **삼장**: 회복/정화/봉인형, 아군 칸 보호
- **저팔계**: 전열 몸빵형, 넓은 몸통 공격
- **사오정**: 연타/반격/물살형, 줄 단위 공격
- **여울**: 매혹/흡수/불길형, 상태이상 압박
- **그 외**: 뚝딱이, 해치, 텐마루, 카와타로, 카게마루

## 개발 원칙

1. 콘텐츠와 런타임 로직 분리
2. 저장 데이터는 복원 가능한 게임 상태만 담음
3. 스토리 연출은 이벤트 명령 시퀀스로 처리
4. 터치 전용, 짧은 세션, 가독성, 실수 복구 우선

## 범위

- 파트 1~16 스토리 라인
- 한국 팀 / 일본 팀 분기 시스템
- 칸 기반 턴제 전투
- 자동/수동 저장 시스템
