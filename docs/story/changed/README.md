# 동유기 변경 파트 설계 메모

이 폴더는 기존 [parts](/Users/kwang/Documents/git-repos/jte/docs/story/parts) 구조를 동유기 버전으로 다시 짠 세부 시나리오 초안이다.

핵심 원칙은 네 가지다.

- 기존 환상서유기의 `분기 구조`, `숨겨진 아이템 조사`, `기믹 던전`, `반복 대화 보상`을 적극 재활용한다.
- 스토리 톤은 정극보다 `해학`, `꽁트`, `민담풍 말장난`에 무게를 둔다.
- 아이템, 스킬, UI/UX도 바꾸지 않고 쓰기보다 `동유기 세계관 언어`로 번안한다.
- 각 파트는 `스토리`, `퍼즐`, `개그`, `재활용 시스템`을 같이 설계한다.

문서 읽는 법:

- `줄거리`: 파트의 큰 사건
- `퍼즐/탐색`: 미로, 기믹, 조사 요소
- `숨은 요소`: 반복 대화, 벽 조사, 조건부 아이템 등
- `꽁트 포인트`: 컷신/필드 대사에서 웃길 지점
- `시스템 재활용`: 원작의 어떤 감각을 가져오는지

이 문서들은 완성 대본이 아니라 `인게임 구현용 설계서`에 가깝다.
