# 동유기 몬스터 기획

## 방향

- 몬스터는 `지역의 농담`과 `지역의 문제`를 같이 보여줘야 한다.
- 잡몹도 그냥 전투용이 아니라, 맵과 파트 분위기를 설명하는 장치여야 한다.
- 기존 `MONSTER.md`의 수치/행동 템플릿은 재활용하고 이름과 설명을 바꾼다.

## 몬스터 재미 포인트

- 장터/축제 파트 몬스터는 말풍선과 행동이 웃겨야 한다.
- 한국 파트 몬스터는 `안개`, `혼례`, `그림자`, `무녀` 이미지를 밀어야 한다.
- 일본 파트 몬스터는 `연무`, `부적`, `수로`, `기계적 질서`가 보여야 한다.
- 봉래산 몬스터는 앞선 지역 속성이 뒤섞여 있어야 한다.

3. 보스 10종 이름/설명 확정
4. 빈 슬롯 226~229 신규 몬스터 설계
