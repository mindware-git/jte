# 동유기 스킬 기획

## 방향

- `SKILL.md`의 범위, 타격감, 버프/디버프 기능은 최대한 유지한다.
- 대신 기술명은 동유기의 캐릭터성과 지역색이 먼저 느껴지도록 바꾼다.
- 기술명만 바꿔도 전투의 분위기가 크게 달라진다.

## 공통 원칙

- 손오공은 시원한 봉술과 분신 계열.
- 삼장은 치유와 봉인, 진언 계열.
- 저팔계는 먹성, 몸빵, 들이받기 계열.
- 사오정은 물살, 연타, 반격 계열.
- 여울은 매혹, 여우불, 생기 흡수 계열.
- 뚝딱이는 난장, 장단, 허풍 버프 계열.
- 해치는 심판, 법화, 적화 계열.
- 텐마루는 검풍/비상/일격 계열.
- 카와타로는 수로/물접시/미끄럼 기믹 계열.
- 카게마루는 연막, 표창, 그림자베기 계열.

## 전투 재미 포인트

- 뚝딱이는 `버프를 거는 캐릭터`가 아니라 `판을 어지럽히는 광대`로 느껴져야 한다.
- 여울은 단순 마법사가 아니라 `홀리고, 빼앗고, 태우는` 타입이어야 한다.
- 해치는 느린 대신 `맞으면 아픈 심판형`으로 차별화한다.
- 카게마루는 닌자답게 `보이지 않게 준비하고 크게 벤다`.
