# 동유기 박스 기획

## 방향

- `BOX.md`는 결국 좌표 데이터다.
- 지금 단계에서는 좌표보다 `맵별 보상 설계`와 `숨김 방식`을 먼저 정하는 게 중요하다.
- 동유기의 박스는 `정직한 상자`, `한 번 더 조사해야 나오는 상자`, `웃긴 보상`, `다음 파트 떡밥`의 네 종류가 섞여야 한다.

## 배치 원칙

- 마을: 노출 상자 2개, 숨은 조사 보상 1개, 개그 보상 1개
- 필드: 회복 상자 2~4개, 장비 상자 1개, 힌트 상자 1개
- 던전: 필수 진행 아이템 1개, 강한 보상 1개, 개그 보상 1개, 숏컷 뒤 보너스 1개
- 장기 던전: 상자 개수보다 `돌아가서 먹는 맛`이 중요

## 숨김 방식 기획

### 1. 대사 연동형

- 같은 NPC에게 여러 번 말을 걸어야 힌트를 준다.
- 예: 장터 할머니가 세 번째 대사에서만 `그 병풍 뒤를 봐라` 같은 말을 한다.

### 2. 벽/장식 조사형

- 병풍, 불상 뒤, 가면걸이, 장독, 갈대뭉치처럼 동유기다운 사물을 조사하게 만든다.

### 3. 조건 해금형

- 웃음 미션 3개 완료 후 상자 생성
- 제단 셋 중 둘 이상 방어 성공 시 보너스 상자 생성
- 눈가림끈을 착용해야 보이는 상자 같은 특수 기믹도 고려 가능

### 4. 되돌아가기형

- 숏컷을 열고 다시 와야 먹는 상자
- 탑에서 내려갈 수단을 확보한 뒤 회수하는 상자
